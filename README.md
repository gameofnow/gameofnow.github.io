# Game of Now

A one page website. A threshold. Nothing more.

## What this is

A single HTML page for [gameofnow.com](https://gameofnow.com) — built to be as simple and open as possible. No tracking. No analytics. No database. No login. No framework. Just a page.

## What it contains

- A sigil
- Five lines
- A Signal link
- A link to download Signal

## How to use it

1. Clone or download this repository
1. Place your `sigil.png` in the same folder as `index.html`
1. Upload to GitHub Pages or any static host
1. Point your domain at it

## Philosophy

Built in the spirit of the gift. Take it, change it, use it, give it away.
